Try These Prompts
Run each prompt in order to light up the entire lifecycle chart
Step 1 — Start a Claude Session
Open a terminal in the demo/ folder and launch Claude
$ cd demo/ && claude
Setup SessionStart InstructionsLoaded
These fire automatically when Claude initializes and reads CLAUDE.md
Step 2 — Ask a Question That Uses Tools
This triggers the core agentic loop: tool request, permission, execution
> What is the weather in Karachi?
UserPromptSubmit PreToolUse PermissionRequest PostToolUse Stop Notification
When Claude calls WebSearch or Bash, the agentic loop hooks fire in sequence
Step 3 — Launch a Subagent
Spawning an agent triggers the subagent lifecycle hooks
> Use Explore agent to read the README.md file
UserPromptSubmit PreToolUse PostToolUse SubagentStart SubagentStop TaskCompleted Stop
The Explore agent runs in a sub-process, firing SubagentStart/Stop around it
Step 4 — Trigger a Failure + Compact
A failing command triggers PostToolUseFailure; /compact triggers compaction hooks
> Run the command: ls /nonexistent/path/xyz > /compact
PostToolUseFailure PreCompact PostCompact
Step 5 — End the Session
Exiting Claude fires the session cleanup hook
> exit or press Ctrl+D
SessionEnd
